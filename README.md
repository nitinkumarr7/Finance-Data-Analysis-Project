# Finance-Data-Analysis-Project
Financial Performance Analysis Dashboard

📊 Project Overview
This project presents an interactive Power BI dashboard designed to visualize and analyze financial data. By transforming raw data into actionable insights, this dashboard evaluates business performance across various segments, products, and geographical regions to support data-driven decision-making.

🛠️ Data Engineering & Processing
The data was sourced from Kaggle and underwent a rigorous cleaning process using Power Query Editor to ensure accuracy and consistency.
Data Cleaning: Removed errors and inconsistencies to ensure a reliable dataset.
Transformation: Corrected date formats and optimized table structures for efficient reporting.

📈 Key Insights & Dashboard Features
The dashboard provides a multi-page view of business health, including:
Financial Metrics: Tracking Total Sales, Total Profit, Units Sold, and YTD performance.
Segment & Product Analysis: Identification of top-performing segments and products (e.g., Paseo as a top-performer).
Trend & Regional Analysis: Visualization of monthly sales fluctuations and country-wise profit distribution to identify strong and weak performing regions.
Operational Metrics: Analysis of discount impacts and manufacturing costs to optimize business operations.

💡 Conclusion
The analysis highlights that Government and Small Business segments are major contributors to total sales. This project demonstrates proficiency in data transformation and the ability to derive meaningful business insights, serving as a foundation for further advanced analytics and predictive modeling.
